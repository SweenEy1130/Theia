User Guide

Use key w to going forward, key s to going backword,key a to going left, key d to going right, and use mouse to rotate view.

Path Tracing

Shoot a ray from each pixel Translate the ray direction from camera space to world space by rtrans matrix and rotate matrix

Water Simulation

Use displacement map to simulate water.
